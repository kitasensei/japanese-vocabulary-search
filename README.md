# いろどり単語検索

『いろどり 生活の日本語』の学習を補助するために作成した、無料の非公式単語検索ツールです。

教科書に掲載されている日本語を入力すると、読み方、掲載課、英語訳などの該当データを検索できます。
スマートフォンとパソコンの両方で利用できます。

## 公開ページ

https://kitasensei.github.io/japanese-vocabulary-search/

## 主な機能

- 日本語・ひらがなによる部分一致検索
- 該当するデータを重複も含めて全件表示
- 読み方、掲載課、英語訳などをカード形式で表示
- スマートフォン対応
- インストール、登録、ログイン不要

## 利用目的

本ツールは、『いろどり 生活の日本語』を使った授業および個人学習を補助する目的で、無償公開しています。
営利目的の教材・サービスではありません。

## データについて

検索データは `data.csv` に保存されています。
CSVを更新することで、検索ページの内容も更新できます。

現在の基本列は次のとおりです。

```text
日本語,ひらがな,掲載課,英語,タイ語,タイ語読み
```

空欄の項目があっても検索できます。
